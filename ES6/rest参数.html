函数的length属性，不包括 rest 参数。

(function(a) {}).length  // 1
(function(...a) {}).length  // 0
(function(a, ...b) {}).length  // 1

注意，rest 参数之后不能再有其他参数（即只能是最后一个参数），否则会报错。

ES6 引入 rest 参数（形式为“...变量名”），用于获取函数的多余

    function add(...values) {
        let sum = 0;
        for (var val of values) {
            sum += val;
        }
        return sum;
    }

add(2, 5, 3) // 10
